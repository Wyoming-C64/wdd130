70 IF A$="6" THEN POKE 53281,5
80 IF A$="7" THEN POKE 53281,6

I wrote this program after learning about IF... THEN statements. At this point, I have also just barely grasped the concept of evaluating variables. I also know that register location 53281 controls the color of the background. For some reason, it did not occur to me to use a numeric variable instead of a string for the user input.

This program evaluates input only for the first 7 colors of the 16-color pallet.
